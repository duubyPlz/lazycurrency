# lazycurrency

AUD <-> TWD/MYR/HKD (mainly for TWD) for the lazy.

- Updates conversion rate if online (fixer.io api)
- If offline, uses last known conversion rate

---

## Setup steps

- `npm install`
- `npm start` to start dev
